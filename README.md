# Галерея фотографий

### Стэк:
* CSS
* HTML
* JavaScript

## Функциональность:
* галерея №1 - клик - карточка увеличивается по ширине, остальные сжимаются в пределах контейнера
* галерея №2 - наведение мышки - карточка увеличивает по ширине и высоте, соседние карточки смещаются в лево
